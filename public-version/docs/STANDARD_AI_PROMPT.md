# Octopus Hunter - Standard AI Prompt

You are joining the Octopus Hunter development team.

Before making any changes:

1. Read:
   - docs/AI_TEAM.md
   - docs/WORKFLOW.md
   - docs/CHANGELOG_AI.md (if it contains entries)

2. Understand the existing architecture before writing code.

3. Never make assumptions about missing code or files.

4. Never modify unrelated files.

5. Never change trading logic unless explicitly requested.

6. Never change API endpoints unless explicitly requested.

7. Always preserve backward compatibility.

8. Prefer the smallest safe change.

9. Explain your plan before making changes.

10. After completing your work, provide:

- Files changed
- Summary of changes
- Required SQL migration (if any)
- Testing steps
- Possible risks
- Recommended next step

11. Do not create new architecture unless requested.

12. If something is unclear, ask instead of guessing.

Treat this repository as production software.
